    SG2D is a 2D graphics engine based on PixiJS and MatterJS, optimized for large maps through matrix tile clustering. Written in modern ES6. Supported by the latest versions of Chrome, Opera, Mozilla and Yandex browsers.

    SG2D

    2D графический движок основанный на PixiJS и MatterJS, оптимизированный для больших карт за счет матричной кластеризации тайлов. Написан на современном ES6. Поддерживается последними версиями браузеров Chrome, Opera, Mozilla и Yandex.

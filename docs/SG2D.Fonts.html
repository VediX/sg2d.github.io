Methods

(static) addFont(options)

Регистрирует шрифт. Как правило добавление шрифтов выполняется после парсинга графических ресурсов (см. SG2D.Utils.parseTexturePack)

Parameters:
Name Type Description
options object
Properties
Name Type Description
name string

Имя для регистрируемого шрифта
schema string

Набор символов в порядке следования в ассете
texture string

Имя базовой текстуры из которой собирается набор графических символов, например, "metal_yellow_big_48x50h". Размер букв определяется из имени текстуры (48x50).
Source:
  • sg2d-fonts.js, line 34
Example
SG2D.Fonts.addFont({
	name: "metal_yellow_big",
	texture: "fonts/metal_yellow_big_48x50h",
	schema: "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789?!().:"
});

### set(name, value, flags = 0, options)

Задать значение свойства

* name
* val
* flags	- допустимые флаги:
	* `SG2G.FLAG1` - ыыы
	* `SG2D.FLAG2` - ssss
* options
	* p1 - sss
	* p2 - rrr
